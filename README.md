# dotfiles
Repo to manage my dotfiles on different computers.

## Install
To install run the following commands:

## With HTTPS
```
git clone --recursive git@github.com:konmenel/dotfiles.git ~/.dotfiles
cd .dotfiles
./install
```

## With SSH
```
git clone --recursive git@github.com:konmenel/dotfiles.git ~/.dotfiles
cd .dotfiles
./install
```
